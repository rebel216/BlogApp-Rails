## BlogAppOnRails

> The Blog app is a classic example of a blog website. It's a functional website that shows the list of posts and empower readers to interact with them by adding comments and liking post.

## Built With

- Ruby
- Rails
- Postgresql

## Getting Started

To run this project, take a copy of the code and follow the instruction below.

### Prerequisites

You need Ruby v3+, Rails v7+ and Postgresql installed on your machine.

### Setup

`Clone the project`

Install gems with:

```
bundle install
```

Setup database with:

```
rails db:create
rails db:migrate
```

### Usage

Start server with:

```
rails server
```

Visit http://localhost:3000/ in your browser.

### Run tests

Install npm with:

Install rspec with:

```
bundle install
```

and

```
rails generate rspec:install
```

run the test with:

```
rspec spec
```

### Open API documentation

```
rails server
```

Visit http://localhost:3000/api-docs in your browser.

### Usage

- Run `rails server` to run the app.
